Introduction:¶

Credit cards are a popular and convenient way to make purchases within the United States. They can be used to manage finances and gather points. However they can also be a source of debt if used incorrectly. Credit cards accumulate money that needs to be paid back to the company/bank. If the statement bill is not paid for in full every month the money will acquire interest on the remaining balance and debt will accumulate. Many Americans have an unwise spending habit where they spend more than they earn. By using this data about credit card user statistics and their debt, we can use data science to make predictions and insights on credit card users.

The data set I have chosen comes from the online source Kaggle which stores numerous datasets publicly online. This data set will provide columns such as gender, age, debt, married, customer, industry, ethnicity, yearsEmployed, Prior default, Employed, credit score, drivers license, citizenship, zip code, income, and approved. Some columns may be confusing at first glance so here is a more detailed synopsis for those columns. Married will be using a binary indicator where 0 will be single/divorced, while 1 would indicate married. The bank accounts column will have a similar indicator where 0 = does not have a bank account and 1 = does own a bank account. The industry column will be the industry of the individual's most recent job. The prior default column will have a binary indicator as well. 0 for no prior defaults and 1 for prior defaults. The employed section will also have a binary indicator 0 = not employed and 1 = employed. Here is a link to the data set to download.

I will utilize many python libraries such as pandas, numpys, matplotlib, seaborn, and etc.

Part 3: Finding the Residuals and utilizing Violin Plots / Adding and Tidying Some Data¶

Using the model coefficient and the model intercept we achieved in part 2, we can now find the expected value for each point when comparing age to debt amounts. I will call this new column the ‘residual’ value. This residual value can be either positive or negative and allows us to create very useful violin plots that can be very insightful when interpreting data.

The expected value basically means the expected debt value of the individual at a certain age. I will gather this information by subtracyting the actual value to the expected value. I will utlize the itterows() function here to go through each row in the dataframe and will also use the panda's .at() function to set values for the new column.

In [ ]:
#How to make a violin plot of residuals vs. Age for the linear model.
#Go through each row and keep track of the index

for index, row in data.iterrows():
  #Find the expected value by taking year and multiplying the Age by the coef and adding the y-int
  ex = data.at[index, 'Age'] * model.coef_ + model.intercept_
  data.at[index, 'Expected_value'] = ex
  #set new column titled residual to actual value - expected
  data.at[index, 'residual'] = data.at[index, 'Debt'] - ex

#plot using the seaborn features
for index, row in data.iterrows():
  data.at[index, 'Age_basic'] = int(data.at[index, 'Age'])
